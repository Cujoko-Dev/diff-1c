Утилита для сравнения md- и ert-файлов.

Поддерживается только KDiff3. В системе контроля версий в качестве программы для просмотра изменений для md- и ert-
файлов нужно указать строку "<Путь до интерпретатора Python> <Путь до данного скрипта> <Обозначение первого файла> 
<Обозначение второго файла>", например 
'"c:\Python34\python.exe" "d:\_Работа\Инструменты\1С\Скрипты\diff-1c-77\diff-1c-77.py" %base %mine' для TortoiseGit.
